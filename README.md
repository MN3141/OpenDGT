# OpenDGT
Repository for implementing communication between DGT 3000 Chess Clock and PC.

## Depencies
### Main documentation
In order to be able to compile the bachelor thesis files (which shall serve as the main documentation) you would need a LaTeX compiler. The development of this project has been done using `TeXLive` package.

### UML diagrams
In order to compile the UML files written in PlantUML you would need the following tools:
- Java Runtime Environment
- [PlantUML JAR file](https://plantuml.com/download)
